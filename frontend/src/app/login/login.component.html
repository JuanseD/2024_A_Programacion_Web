Bienvenido a Génesis.

Génesis es el sistema para gestión de servicios estudiantiles tales como Matrícula, Horarios, Plan de estudios, Calificaciones y Certificados, entre otros. Está construido por la Universidad de Santander con tecnologías Web de punta y optimizado para la gestión de grandes cantidades de datos.

Génesis se integra con la aplicación para la gestión de Horarios (Master), con la aplicación para la gestión de servicios docentes (Galileo) y con la aplicación para la gestión de servicios de Registro y Control Académico (Davinci). Estas aplicaciones conforman el sistema integral de gestión académica (SIGA).
